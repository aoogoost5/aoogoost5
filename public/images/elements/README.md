# 五行元素符号

此目录用于存放五行元素的符号图像。在MVP阶段，我们只需要每种元素的基础符号。

## 预期文件

- `wood-symbol.png` - 木元素符号
- `fire-symbol.png` - 火元素符号
- `earth-symbol.png` - 土元素符号
- `metal-symbol.png` - 金元素符号
- `water-symbol.png` - 水元素符号

## 设计规范

- 尺寸：512x512像素
- 格式：PNG（透明背景）
- 风格：简约、现代、符合五行传统色彩

## 注意事项

这些符号将用于应用的核心功能，应确保设计清晰、辨识度高。
